# Hotel Booking Frontend

Современный фронтенд для системы бронирования отеля, построенный на React + TypeScript + Tailwind CSS + shadcn/ui.

## Технологии

- **React 18** с TypeScript
- **Vite** - сборщик
- **Tailwind CSS** - стилизация
- **shadcn/ui** - UI компоненты
- **React Router** - маршрутизация
- **TanStack Query** - управление серверным состоянием
- **Zustand** - глобальное состояние
- **Axios** - HTTP клиент
- **React Hook Form + Zod** - формы и валидация
- **date-fns** - работа с датами

## Установка

```bash
npm install
```

## Настройка

Создайте файл `.env` в корне проекта:

```env
VITE_API_URL=http://localhost:8081/api
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

## Сборка

```bash
npm run build
```

## Роли пользователей

### CLIENT (Клиент)
- Просмотр доступных номеров
- Поиск и фильтрация номеров
- Создание бронирований
- Просмотр своих бронирований
- Управление профилем

### OPERATOR (Оператор)
- Просмотр всех бронирований
- Check-in и check-out гостей
- Фильтрация бронирований по статусу

### ADMIN (Администратор)
- Управление номерами (CRUD)
- Управление типами номеров (CRUD)
- Управление гостями
- Просмотр всех бронирований
- Просмотр статистики

## API интеграция

Все API запросы настроены для работы с бэкендом на `http://localhost:8081/api`.

JWT токен автоматически добавляется в заголовки запросов через Axios interceptor.
